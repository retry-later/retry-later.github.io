C语言-数据类型

Author: pm&fc Date: 四月 24, 2022  23:54:20 Category: C语言

数据类型

  • 变量的定义
  • 变量的赋值
  • 进制

  • 程序运行完等待结束的方式

    • 设置断点
    • system(“pause”);
    • getchar();

  • 整数类型的范围

    • int 32bit（在较老的计算机上只能存储16位）
    • short int 16bit 短整型 （要求是不长于int类型） %hd
    • long int 32bit （不短于int类型） %ld （如果一个长整型的数组使用十六进制进行打印的话可以使用%lx 来表示，八进制就是%lo）
    • long long int 64 bit %lld
    • unsiged [int/long int/ short int /long long int] 无符号整型也就是没有负数 %u

  • 字符类型

    • char 理论上占8bit，但是实际上并不是。比如Unicode

  • 浮点类型

    • float 单精度 32bit %f
    • double 双精度浮点类型 64bit %f
    • 浮点寄存器其实并不能精确的存储

  • 布尔类型（C99标准引入）

  • 总结
