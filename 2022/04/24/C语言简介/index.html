C语言扫盲

C语言简介

  • C语言创始人：丹尼斯·里奇
  • C语言特点：
    • 高效性
    • 可移植性
    • 灵活性
  • 编程语言的分类
    • 低级语言
      • 机器语言 （01二进制字符串）
      • 汇编语言
    • 高级语言
      • 面向过程
        • C语言
      • 面向对象
        • 编译型语言 -> C++ （将源代码编译为机器码供CPU直接执行）
        • 解释型语言 -> Python（需要一个解释器翻译为计算机能够读懂的语言）

C语言标准

    2. C语言标准的历程

数据类型

整数类型

  • 声明变量

    • int num

  • 批量声明变量

    • int num1,num2,num3

  • 声明赋值

    • int num1=1
    • int num1,num2 = 5 //这里有个问题，就是其实只有num2被复制了
    • int num1=5,num2=5,num3=5

scanf函数

  • scanf函数输入函数，但是因为该函数容易引起缓冲区溢出之类的问题，所谓微软提供了一个新的函数。scanf(“%d”,&Num) %d数据类型，Num变量名
  • scans_s 输入函数 scarf_s(“%d”,&Num,10) 建议使用scanf，因为如果跨平台的话可能会产生错误。
  • %d告诉程序接收到的参数是int类型
  • &Num4 告诉程序接收到到的数据存储在Num4中
  • vs2015中如果使用scanf而不使用scanf_s，会触发一个报错。声称这个函数是不安全的，如果我们强制使用该函数的话则需要在项目名–属性–C/C++–预处理器–预处理器定义–编辑– win32前加入 _CRT_SECURE_NO_VARNINGS 即可使用。
