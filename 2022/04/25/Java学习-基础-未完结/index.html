Java的技术体系

Java从源代码到执行的流程

.java 文件（源代码） ——> javac 编译 ——> .class 字节码文件 ——> jvm执行

  • 编写源代码
  • 编译（生成Hello.class，因为输出的里面有中文，可能会因为编码问题导致编译出错，所以我们制定编码进行编译。）

javac -encoding utf-8 Hello.java

  • java执行Hello（注意后面没有.class，因为Java认为要执行的是Hello这个类，它会寻找这个类的mian入口函数；而不是要执行Hello.class这个文件，这个过程就是将Hello装载到jvm虚拟机上进行执行。）

java Hello

关于Java源代码的几点注意事项

  • Java的源文件是以.java为扩展名的文件，基本组成单元为class，也就是一个源代码文件中至少要有一个class。

  • Java的main方法的书写是一个固定的格式 public staic void main(String[] args){}

  • Java中严格区分大小写

  • 源代码中最多只能有一个public 的类，其他类的个数不做限制。

  • 源文件的文件名，必须和public类的类型一致，否则将无法编译。

Java变量

Java是一个强类型的语言，强类型语言有两个含义：

  1. 变量必须先声明再使用。

  2. 指定类型的变量只能接受该类型的值。

定义变量

Java中可以通过指定数据类型和标识符来申明变量如下：

DataType identifier;

或

DataType identifier = value ;

上述语法中涉及到了三个内容 DataType、identifier和value，分别为变量类型（int、double、char、string等）、变量名（标识符）和赋予的值。

变量初始化

上面我们只是对变量进行了声明但是我们还需要给变量一个明确的初始值，给变量设置初始值的方式有两种，一种是声明时进行赋值，另一种是先声明后赋值。

关于初始化的两点注意事项：

  1. 变量是类或者结构中的字段，如果没有显示的初始化则声明时默认初始化的值为0；
  2. 方法中的变量必须初始化，否则使用变量的时候会报错。

变量的作用域

​ 变量的作用域规定了能够使用某个变量的范围，只有在范围内才能够使用变量。变量的声明地点不同，作用域也不同。根据作用域的不同变量分为成员变量和局部变量。

成员变量

​ 成员变量分为全局变量和静态变量，定义在方法体和语句块之外，不属于任何一个方法，作用域是整个类。两种成员变量的区别在于：

名称 修饰符 访问 生命周期
全局变量（实例变量） 无static修饰 对象名.变量名 只要对象被当作引用，实例变量就将存在
静态变量（类变量） 有static修饰 类名.变量名或对象名.变量名 其生命周期取决于类的生命周期。类被垃圾回收机制彻底回收时才会被销毁

局部变量

​ 局部变量是指在方法或者方法代码中定义的变量，其作用域是在是所在的代码块中。总体可以分为三种：

  • 方法参数变量（形参），作用域为本方法体。
  • 方法中定义的局部变量，作用域为定义变量的位置到方法结束的位置。
  • 代码块中定义的局部变量，作用域为定义变量的位置到代码块结束的位置。

局部变量在使用之前必须被赋值

浮点类型

​ 浮点类型是带小数部分的类型，也叫实型。浮点类型数据包括单精度浮点型（float）和双精度浮点类型（double），代表着有小数精度要求的数据。

​ 单精度浮点类型和双精度浮点类型之间的区别是所占用的内存空间大小，float占4个字节的空间，而doubel占8个字节的空间，所以double比float所表示的范围更大。Java中默认的浮点类型是double，如果要声明一个float类型需要在其值的后面加上一个f/F比如 1.23f/1.23F表示float类型。

字符类型

​ Java中的字符类型使用两个字节的 Unicode 编码表示，他支持世界上的所有语言，使用单引号包含字符或者整数对char类型赋值。一般计算机使用ascii码，使用一个字节表示一个字符，Ascii码是unicode码的一个子集，用unicode表示ascii时高字节为0，Unicode 字符通常用十六进制表示。例如“\u0000”~“\u00ff”表示 ASCII 码集。“\u”表示转义字符，它用来表示其后 4 个十六进制数字是 Unicode 码。

Java中的类型转换

​ 数据类型转换发生在所赋值的数据类型和接收的数据类型不一致的情况下，它需要从一种数据类型转换到另一种数据类型。数据类型转换分为隐式转换（自动类型转换）和显式转换（强制类型转换）。

隐式转换需要满足以下两个条件：

  • 两个类型彼此兼容
  • 目标类型取值范围大于原数据类型

显示转换：当两种数据类型不兼容，或目标类型的取值范围小于源类型时，自动转换将无法进行，这时就需要进行强制类型转换。

语法为： (type)variableName

Java中的直接量

​ 直接量是指在程序中通过源代码直接给出的值，类如在 int a = 5 ;代码中，为变量a所分配的初始值5就是一个直接量。并不是所有的数据类型都有直接量的，能够制定直接量的类型大致分为三种：基本类型（int、long、float、double……）、字符串类型和null类型,null 类型是一种特殊类型，它只有一个值：null。而且这个直接量可以赋给任何引用类型的变量，用以表示这个引用类型变量中保存的地址为空，即还未指向任何有效对象。具体支持的八种类型有：

  • int 类型的直接量
    在程序中直接给出的整型数值，可分为二进制、十进制、八进制和十六进制 4 种，其中二进制需要以 0B 或 0b 开头，八进制需要以 0 开头，十六进制需要以 0x 或 0X 开头。例如 123、012（对应十进制的 10）、0x12（对应十进制的 18）等。
  • long 类型的直接量
    在整型数值后添加 l 或 L 后就变成了 long 类型的直接量。例如 3L、0x12L（对应十进制的 18L）。
  • float 类型的直接量
    在一个浮点数后添加 f 或 F 就变成了 float 类型的直接量，这个浮点数可以是标准小数形式，也可以是科学计数法形式。例如 5.34F、3.14E5f。
  • double 类型的直接量
    直接给出一个标准小数形式或者科学计数法形式的浮点数就是 double 类型的直接量。例如 5.34、3.14E5。
  • boolean 类型的直接量
    这个类型的直接量只有 true 和 false。
  • char 类型的直接量
    char 类型的直接量有三种形式，分别是用单引号括起来的字符、转义字符和 Unicode 值表示的字符。例如‘a’，‘\n’和‘\u0061’。
  • String 类型的直接量
    一个用双引号括起来的字符序列就是 String 类型的直接量。

在大多数其他语言中，包括 C/C++，字符串作为字符的数组被实现。然而，在 Java 中并非如此。在 Java 中，字符串实际上是对象类型。在教程后面你将看到，因为 Java 对字符串是作为对象实现的，因此，它有广泛的字符串处理能力，而且功能既强又好用。
8）null 类型的直接量
这个类型的直接量只有一个值，即 null。

Java中对于字符串类型需要注意的

​ 当程序第一次使用某个字符串的直接量时，Java会使用常量池来缓存该字符串的直接量，如果程序后面的部分需要用到该字符串的直接量时，Java会直接使用常量池中的字符串直接量。

​ 常量池是指在编译期间被确定并且保存在已编译的class文件中的一些数据，包含了类、方法、接口中的常量、也包括字符串直接量。

​ 在Java中Java会保证每个字符串常量只有一个，不会产生对个副本。所以s1 = "this is a test!"; s2 = "this is a test!"中s1和s2是同一个对戏那个，s1 == s2 得出的值为true。

计算机中的原码、反码、补码

对于有符号而言：

  • 二进制的最高位是符号位：0表示正数、1表示负数
  • 正数的原码、反码、补码都一样（三码合一）
  • 负数的反码 = 它的原码符号位不变，其他位取反
  • 负数的补码=它的反码+1，负数的反码 = 负数的补码 - 1
  • 0 的反码、补码都是0
  • Java没有无符号数，换而言之，Java中的数都是有符号的。
  • 在计算机运算中，都是以补码的方式来运算的。
  • 当我们看运算结果的时候，要看它的原码

根据计算机运算中都是以补码的方式进行运算的，所以Java中的二进制位运算（& | ~ ^ )都是按照操作数的补码进行计算的。下面的例子以按位与和按位取反两种位运算为例，另外两种按位运算原理一样：

>> 、<<、>>>

  • 算数右移 >>： 低位溢出，符号位不变，并用符号位补溢出的高位
  • 算数左移<< : 符号位不变，低位补0
  • 逻辑右移（无符号右移）>>> : 低位溢出，高位补0 （不管正负标志位为0还是1，将该数的二进制码整体右移，左边部分总是以0填充，右边部分舍弃。）

分支结构switch的一些细节

  • 表达式的数据类型应该和case后的常量类型一致，或者是可以自动转换成可以相互比较的类型。比如输入的是字符，而常量是int。
  • switch(表达式)中表达式的返回值类型必须是(byte,short,int,char,enum,String)中的，否则会报错。比如switch(sex)中的sex的数据类型必须是上述类型之一。
  • case子句中的值必须是常量，而不能是变量。
  • default子句式可选的，
  • break语句用来在执行完一个case分支之后跳出switch语句，如果没有写则会一直运行到switch语句块的结尾。
